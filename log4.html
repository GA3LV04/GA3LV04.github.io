How CSS Uses Color

CSS uses color to control the appearance of text, backgrounds, borders, and other visual elements on a webpage. By applying different color values to specific properties, designers can create contrast, hierarchy, branding, and visual interest.

How does CSS accept color as a value?

CSS allows colors to be written in several different formats.

Named Colors

These are predefined color names built into CSS.

An example of this: red, blue, green, white, etc.

Hexadecimal (Hex) Colors

Hex colors start with # and use six characters (numbers 0–9 and letters A–F).

Format: #RRGGBB RR = Red value GG = Green value BB = Blue value

RGB Colors

RGB stands for Red, Green, Blue. Each value goes from 0 to 255.

Format: rgb(red, green, blue)

Other Formats

RGBA (RGB with Alpha / Transparency) The “A” stands for alpha (opacity), which ranges from 0 (transparent) to 1 (fully visible).

This creates a semi-transparent red background.
